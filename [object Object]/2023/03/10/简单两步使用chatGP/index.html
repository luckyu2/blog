简单2步使用chatGPT

准备工作：需要自备 OpenAI API Key

AMA 中文名问天，是一款第三方的 ChatGPT 客户端，简单易用，支持自定义 API，需要自备 API Key。

使用方法：

1.下载应用

1.1安卓用户需要下载应用APP，下面地址任选

蓝奏下载地址

阿里云下载

1.2 苹果用户需要到应用商店下载TestFight，打开下面链接
https://testflight.apple.com/join/umM4Sg9F
加入 Beta 版“AMA”

2.输入KEY，开始使用

打开应用在界面输入你的的key，注意无魔法上网的需要选择lnkcast，即可开始使用。
